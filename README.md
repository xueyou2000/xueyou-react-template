# React项目模板

> 注意 `lint-staged` 只会检查 git 暂存区(staged)的文件，而不是所有文件。如果历史文件有lint错误，请手动执行`npm run lint`
> 并且 `lint-staged` 会自动尝试修复代码, 比如格式化

本模板使用技术(使用最新技术):

- 构建： Turborepo, RsBuild, PNPM多仓管理
- 前端框架: React V19
- 性能优化: React Scan
- 状态管理: Zustand
- 路由: React Router V7
- 样式: SCSS
- 图标: React Icons
- 代码规范: ESLint, Stylelint, Prettier, Husky, Commitlint, Branchlint
